# Real-Time Lecture Assistant

Capture live audio ➔ transcribe ➔ polish ➔ TTS ➔ play back in real-time.

## Setup

1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

2. Add your API keys:
    - OpenAI Whisper
    - Gemini 1.5 Pro
    - ElevenLabs TTS

3. Run:
    ```bash
    python main.py
    ```
